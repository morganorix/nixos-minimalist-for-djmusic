# nixos-minimalist-for-djmusic
A minimalist linux based on NixOs. It just use hyprland with nemo, rofi, foot, fish, nvim, zen browser, mixxx. It consum 820Mo of RAM. The processors is always in performance. Allpower and zero latences ;-)
